## ğŸ¯ About

**Ustaad (E-Mazdoor)** is Pakistan's pioneering **full-circle recruitment and service platform** that revolutionizes how people connect with work opportunities. We bring together daily laborers, skilled professionals, high-level executives, freelancers, companies, households, and service seekers under one unified digital roof.

### ğŸŒ The Problem We Solve

Pakistan's labor market faces significant challenges:
- **Fragmented Market**: Daily wage workers struggle to find consistent work
- **Limited Access**: Skilled professionals lack platforms to showcase their talents
- **Trust Issues**: Households and companies struggle to find reliable service providers
- **Information Gap**: No centralized platform connecting all workforce segments

### ğŸ’¡ Our Solution

Ustaad (E-Mazdoor) bridges these gaps by providing:
- A **unified platform** for all workforce categories
- **Direct connections** between service providers and seekers
- **Verified profiles** ensuring trust and reliability
- **Fair opportunities** for everyone, from daily wage workers to CEOs
- **Digital empowerment** for Pakistan's workforce

## ğŸš€ Getting Started

### Prerequisites

Make sure you have the following installed:
- **Node.js** 18.17 or later
- **npm** or **yarn** or **pnpm** or **bun**
- **Git**
- **PostgreSQL** (for database)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/taha-zaidii/Ustaad-E-Mazdoor.git
   cd Ustaad-E-Mazdoor
   ```

2. **Install dependencies**
   ```bash
   npm install --legacy-peer-deps
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   ```
   
   Configure your `.env.local` file with:
   ```env
   DATABASE_URL=your_postgresql_connection_string
   NEXT_PUBLIC_APP_URL=http://localhost:3000
   # Add other environment variables as needed
   ```

4. **Set up the database**
   ```bash
   # Run database migrations (when available)
   npm run db:migrate
   ```

5. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

6. **Open your browser**
   
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

### Code Style & Conventions

- **Components**: Use functional components with TypeScript
- **Naming**: PascalCase for components, camelCase for functions/variables

- **Imports**: Use absolute imports with `@/` prefix
